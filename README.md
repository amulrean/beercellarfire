# Beercellarfire

## Dev Setup

- Using the template src/environments/environment-secrets.ts-template Create src/src/environments/environment-secrets.ts with Fiberbase keys filled in.
- $ npm install
- $ ng serve

## Firebase Hosting

```
$ npm install -g firebase-tools
$ firebase login
$ firebase init
$ firebase deploy
```
